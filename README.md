# Appointment Queue Management System

**FYP Project - Final Year Project Submission**

A comprehensive Laravel 12 application for managing dental clinic appointments with real-time queue visualization and patient tracking.

---

## ðŸŽ¯ Project Overview

This system provides:

- **Patient Booking:** Online appointment scheduling with available slot management
- **Check-In Management:** Real-time patient check-in with queue position tracking
- **Queue Management:** FIFO queue system with real-time updates
- **Patient Tracking:** Track appointment status and queue position via unique token
- **Staff Dashboard:** Live queue board with real-time updates (polling-based, 2-second intervals)
- **Feedback System:** Automated post-treatment feedback collection
